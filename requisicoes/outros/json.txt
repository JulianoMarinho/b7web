==========================JSON PARSE===============================

JSON.parse(); ele pega uma string de json e a transforma em um json real

exemplo: let pessoa = '{"nome": "nome", "idade": 90}';

=============================JSON stringify==========================
Esta função recebe um json e retorna uma string de json, no caso 
é o contrário da json.parse.

========================Síncrono vs Assíncrono=======================
o Código síncrono vai executando linha por linha, esperando a execução de cada linha para avançar
o Código assíncrono vai executando linha por linha e não espera se a linha foi executada para executar a próxima

Toda requisição interna, é uma requisição assíncrona.
